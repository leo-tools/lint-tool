# taobao-mp-tool

## 安装
```bash
npm install -g @leo-tools/tmp
```

## 使用

### 创建小程序
```bash
// 创建淘宝小程序
tmp create taobao-miniapp 

// 创建淘宝小部件
tmp create taobao-livecard

// 创建千牛小程序
tmp create qianniu-miniapp
```

### 生成页面/组件
```bash
// 创建页面
tmp g p

// 创建组件
tmp g c
```

### 增加 feature
```bash
// 增加 lint
tmp add lint
```
